# Web Design Enhancer V3 (3.0.0a1)

**Principle:** the model proposes and implements; the orchestrator authorizes, verifies, and blocks.

**Stabilization status (Phase 11+ hardening):** core DoD met; public `wde transition` **removed** (domain events only). Evidence envelopes re-verify digests (optional `WDE_EVIDENCE_SECRET` HMAC). Aesthetic delivery bar = **80** + full dimensions; `independent-clone` is **declared** only unless `WDE_ALLOW_DECLARED_INDEPENDENCE=1`. Multi-model medians remain lab work.

Full plan: [`docs/superpowers/specs/2026-07-10-wde-v3-plan.md`](superpowers/specs/2026-07-10-wde-v3-plan.md)

## Creative Discovery pipeline (authoritative)

```text
interpret_request
  → run_all_research          # receipts under .wde/research/
  → synthesize_research       # ResearchSynthesis + coverage report
  → generate_territories      # curated families OR grammar (P6)
  → select_territory          # single winner, no mashup
  → write_contracts           # brief / experience / DESIGN.md / lock
                              # DESIGN.md compiled from winner.tokens
  → decision-graph.json       # source → principle → transform → contract → code → render
  → contract/code/render traces
```

### Territory → DESIGN.md (P0)

Each territory carries structured `DesignTokens` (`mode`, hex roles, fonts).  
`compile_design_md` **must** emit those tokens:

- light winner (e.g. paper ivory `#F3EEE4`) → **Light-first project**, never bare *Dark-first*
- dark winner → Dark-first with winner accent/surface

Regression: `test_light_editorial_territory_does_not_compile_dark_first`.

### Research consumption (P1–P2)

Receipts are not “called for show”. They become:

| Artifact | Role |
|----------|------|
| `research-synthesis.json` | sector / visual / anti-ref / cross-domain findings |
| `research-coverage.txt` | human coverage + confidence + degraded mode |
| territory `anti_references` | folded from synthesis |
| receipt fields | `source_type`, `network_used`, `executor`, `degraded` |

Example coverage block:

```text
Research coverage
✓ Internal sector framing
✓ Local UI corpus
✗ Live visual references unavailable
✗ External cross-domain research unavailable
Confidence: limited
Mode: degraded (no successful external tools)
```

### Receipt integrity (P3)

- `digest` + `artifact_sha256` sealed by `wde-core`
- `request_hash` ties receipts to the discovery request
- `invalidate_receipts_for_request()` marks stale receipts when the request changes
- hand-edited executors outside the allowlist are **rejected**
- tampered artifacts fail `receipt_is_valid`

### Decision graph (P4)

`.wde/discovery/decision-graph.json` links:

```text
source → principle → transformation → contract → code → render
```

### Traces (P5)

| Trace | Proves |
|-------|--------|
| `contract_trace` | signature, metaphor, CTA, tokens across the four contracts |
| `code_trace` | `{id}-signature` / tokens in implementation (deferred if no code) |
| `render_trace` | Playwright: desktop+mobile visibility, interaction, captures under `.wde/discovery/render/` |

### Grammar diversity (P6)

Axes: metaphor × structure × signature × type × materiality × image × motion × conversion  
with compatibility constraints and pairwise structural distance.

```python

```

- Bugatti is **not** a universal metaphor (anti-list enforced)
- Diversity tests: 20 sectors, max metaphor share ≤ 45%
