# Web Design Enhancer V3 (3.0.0a1)

**Principle:** the model proposes and implements; the orchestrator authorizes, verifies, and blocks.

**Stabilization status (Phase 11+ hardening):** core DoD met; public `wde transition` **removed** (domain events only). Evidence envelopes re-verify digests (optional `WDE_EVIDENCE_SECRET` HMAC). Aesthetic delivery bar = **80** + full dimensions; `independent-clone` is **declared** only unless `WDE_ALLOW_DECLARED_INDEPENDENCE=1`. Multi-model medians remain lab work.

Full plan: [`docs/superpowers/specs/2026-07-10-wde-v3-plan.md`](superpowers/specs/2026-07-10-wde-v3-plan.md)

## Install (dev)

```bash
cd web-design-enhancer-pro
pip install -e .
wde --help
# without console script on PATH:
python -m wde --help
# from repo root without install (PYTHONPATH=.):
python wde.py --help
```

## First commands

```bash
wde init                 # .wde/ + brief + experience stubs
wde status --json
wde next                 # always a real public command
wde doctor
wde validate intent
wde validate research    # pillars on disk (search.py + getdesign)
wde validate experience  # UX contract (alias: architecture)
wde validate design
wde validate lock
wde run static
wde run mechanical
wde run visual --url http://localhost:5173
wde deliver-check
wde review --emit-package --url http://localhost:5173
wde review --url http://localhost:5173
wde report               # consolidated JSON + MD under .wde/reports/
wde benchmark            # smoke corpus; never auto-delivers
wde migrate-v2           # import V2 projects safely
# `wde transition` is REMOVED from the public CLI (always errors)
```

## Control plane (per user project)

```text
.wde/
  project.json
  state.json
  capabilities.json
  evidence/
  reports/
  logs/
  cache/
CREATIVE-BRIEF.md
```

`state.json` is owned by **wde-core**. Agents must not hand-edit `phase: READY_TO_DELIVER`.

## Relationship to V2 skill

V2 `SKILL.md` + `scripts/*` remain the validation arsenal. V3 wraps them in a **stateful orchestrator** so models cannot claim gates without fresh proof.

Tagged freeze of V2 line: **`v2.2.0`**.
